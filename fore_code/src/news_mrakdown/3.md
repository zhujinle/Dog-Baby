
# 【我与国家一起前行】“解锁”和美乡村
<span style="color:blue;font-size:15px">央视网</span>
<span style="color:grey;;font-size:15px">&emsp;2024年01月04日 23:06</span><br>

<span style="font-size:17px">&emsp;&emsp;<span style="font-weight:bold">央视网消息</span>（晚间新闻）：我与国家一起前行，把目光投向广袤乡村。2023年，绿水青山成色更足，乡村振兴展现新气象。党的二十大报告提出建设“和美乡村”的目标，从此前的“生态宜居 美丽乡村”升格为“宜居宜业 和美乡村”，中国的乡村应该在哪里升级蜕变，需要每一个具体的村庄寻找最为适合自己的答案。2023年底，记者来到了浙江省德清县，这里正在“解锁”着不一样的诗画田园。<br><br>
&emsp;&emsp;白墙、灰瓦、流水、人家。<br><br>
&emsp;&emsp;这个冬季，江南水乡德清蠡山村，吸引着不断到访的游客。<br><br>
![](https://p3.img.cctvpic.com/photoworkspace/contentimg/2024/01/04/2024010422564879895.png)
<br>&emsp;&emsp;民宿管家忙着接待客人乘船入住。<br><br>
&emsp;&emsp;2023年，26岁的黑龙江姑娘柴秀红作出了人生中的一个重要决定：辞掉在杭州星级酒店的工作，来到蠡山村当民宿管家，同时也成了这里的新村民。<br><br>
&emsp;&emsp;<span style="font-weight:bold">“蠡山沄居”民宿管家 柴秀红：</span>选择了来这边，然后就是选择一个慢生活和一个深文化的地方。因为我本身也是一个很喜欢中式文化和美学中式美学的那种人。<br><br>
![](https://p3.img.cctvpic.com/photoworkspace/contentimg/2024/01/04/2024010422574298686.png)
<br>&emsp;&emsp;喜欢，让不止一个像柴秀红这样的年轻人，在2023年选择就职蠡山村。他们给这个小山村带来了新鲜的理念和创业方式，比如给游客定制专属的乡村游路线、设计一顿仪式感满满的传统中餐，或者拍一套美美的汉服照。<br><br>
&emsp;&emsp;<span style="font-weight:bold">河南游客 魏琰笑：</span>可以让第一次来的游客就能够一站式到底沉浸式的一个体验。我们会更希望去一些比较安静的大自然的，那农村其实就是一个比较合适的一个地方。 <br><br>
<span style="font-weight:bold">&emsp;&emsp;浙江德清：保护为先 擘画宜居宜业山水田园</span><br><br>
![](https://p4.img.cctvpic.com/photoworkspace/contentimg/2024/01/04/2024010422584840640.png)
&emsp;&emsp;“景中有可居住的村、村中有可就业的景”。此前，有山有水的蠡山村就一直在尝试发展旅游经济，村委曾收到过不下百份的合作意向，而他们最终的选择标准是：在宜居和宜业的兼容中“解锁”乡村的和美。 从规划建设到运营管理，力求每个环节都最大限度保护村庄的原有水系生态和文化古迹，同时改善村民的生活环境。最终，“蠡山沄居”项目在村里落地，用 “范蠡西施”典故、古桥古迹等传统文化，打开“乡村+民宿”的新模式，同时参与到村庄的保护中去。<br><br>
![](https://p3.img.cctvpic.com/photoworkspace/contentimg/2024/01/04/2024010422585542402.png)
<span style="font-weight:bold">&emsp;&emsp;“蠡山沄居”创始人 唐仲磊：</span>我做这个项目的一个目的也是为了把中国传统文化保留在当地。这个项目既把古建筑跟旅游、住宿结合在了一起，也希望给当地带来一定新鲜的理念。<br><br>
&emsp;&emsp;新鲜的理念，切实的改变。这里一年四季向村民免费开放，茶余饭后村民都可自由进出、散步游玩。宜居，也要宜业。土地租赁模式为当地村民每年增加超百万的租金收入，很多村民还多了一份家门口的工作机会。在乡村创客中心里，村民们手工制作的精美汉服供不应求。村民范寿山在退休前开公交，如今在家门口的湖里开起了船。<br><br>
![](https://p2.img.cctvpic.com/photoworkspace/contentimg/2024/01/04/2024010422593412778.png)
<br><span style="font-weight:bold">&emsp;&emsp;“蠡山沄居”船夫 范寿山：</span>我今年70岁了，他们招工，招一个船工，那么我来报个名嘛，他们说好的，让我来开船了。<br><br>
范师傅说，村里像他这个年纪的老人，大多都身体硬朗，不少村民都在这个民宿上班，有做船夫的，有打扫卫生的，有烧饭做菜的，一日三餐包吃，每月还赚3000多元。<br><br>
<span style="font-weight:bold">&emsp;&emsp;“蠡山沄居”船夫 范寿山：</span>上海人来到这里，说空气真好，他们问我你几岁了，我70岁，他们不相信的，他们说我长得年轻。<br><br>
![](https://p5.img.cctvpic.com/photoworkspace/contentimg/2024/01/04/2024010423002051961.png)
<br>&emsp;&emsp;立足资源禀赋，做好山水文章，是蠡山村“解锁”和美乡村的方式。<br><br>
&emsp;&emsp;而在2公里外的城北村，转变思路，打开想象，废弃矿坑也可以变成宝藏营地。<br><br>
![](https://p3.img.cctvpic.com/photoworkspace/contentimg/2024/01/04/2024010423004635518.png)
<br>&emsp;&emsp;2023年的冬至这天，全国各地素不相识的人来到这片“看得见稻田的矿坑营地”，一起生火、摘菜、包饺子。<br><br>
<span style="font-weight:bold">&emsp;&emsp;上海游客 才华：</span>摘菜、洗菜，然后自己剁自己包。有带小朋友过来让他们来体验一下。<br><br>
<span style="font-weight:bold">&emsp;&emsp;杭州游客 马程杰：</span>去体验以前的那种家族在一起，这种热闹的一个氛围。<br><br>
<span style="font-weight:bold">&emsp;&emsp;上海游客 李雯雯：</span>正是因为这样一个非常自然的一个乡村，才能让我们大家更愿意聚集在这儿，更接近自然。 <br><br>
<span style="font-weight:bold">&emsp;&emsp;浙江德清：转变思路 废弃矿坑变宝藏营地</span><br><br>
&emsp;&emsp;这里原本既没有稻田也没有营地，只是一座废弃多年的矿坑。政府带着村民复垦，让一千多亩的矿坑地种上了水稻、飘起稻花香，然而还有几百亩不具备复垦条件的矿坑一直裸露着，直到几个创业年轻人和村里一拍即合，在矿坑上改造和美乡村的行动开始了。打造酷炫的“矿坑露营地”，融入露营咖啡、围炉煮茶、传统文化课等文旅消费热点，从“初级农产品产销”到“农文旅融合发展”，曾经寸草不生的矿坑变成了村里的风景线和摇钱树。<br><br>
![](https://p5.img.cctvpic.com/photoworkspace/contentimg/2024/01/04/2024010423012282432.png)
<br><span style="font-weight:bold">&emsp;&emsp;“稻颂乾元”矿坑营地创始人 谢逊：</span>我们每年整个项目要为村集体贡献，村集体的收入要增加，每一年都要增加15万，第四年开始要60万一年。 <br>
&emsp;&emsp;这个项目为当地村民带来了150多个就业岗位。24岁的咖啡师陈尊康就是在这里开启了不一样的人生体验。在他看来，在家门口做好一杯咖啡，服务南来北往的顾客，生活也可以很充实。<br><br>
![](https://p3.img.cctvpic.com/photoworkspace/contentimg/2024/01/04/2024010423021440297.png)
<br><span style="font-weight:bold">&emsp;&emsp;“稻颂乾元”矿坑营地咖啡师 陈尊康：</span>环境好嘛，心情也好了。别看是在乡下，但是接触到有很多（人），生活方式、习惯根本都不一样，也是能学习的。有很多外国人来嘛，他们到这边用的是那种散装中文，我们给他们讲话有这种散装英文的这种感觉。<br><br>
<span style="font-weight:bold">&emsp;&emsp;用“乡村+”打开中国乡村振兴万千气象</span><br><br>
![](https://p3.img.cctvpic.com/photoworkspace/contentimg/2024/01/04/2024010423022279715.png)
<br>&emsp;&emsp;乡村+露营，乡村+民宿，乡村＋生态，乡村＋文化。当你用不同的关键词“解锁”和美乡村，会发现还有一种可能：乡村＋数字。在德清县，通过“数字乡村一张图”实时大屏，城北村的垃圾桶能智能感应垃圾分类情况，曲溪村的渔民打开手机就能看鱼池的溶氧量、pH值、水温，蠡山村实现了全村水系24小时云眼监测，污染或是违规捕鱼，都能第一时间发现。家家户户门口的二维码，扫一扫就能看到公共信息、享受便民服务，还能挂号，而不久后，老人健康监测也将全覆盖。新的一年，更“聪明”的乡村治理和服务将触达生活的各个角落，新老村民也对和美乡村有了更深的理解，更多的期待。 <br><br>
<span style="font-weight:bold">&emsp;&emsp;“蠡山沄居”船夫 范寿山：</span>空气好，我们这里吃得也好，小孩也大了，都成家了，我们两个老的待在家里，开开心心。<br><br>
<span style="font-weight:bold">&emsp;&emsp;“蠡山沄居”民宿管家 柴秀红：</span>美的呢，那主要就是能够看到摸得着的这种，和的话呢主要是一个内在的一个精神的面貌。就比如说和睦、和善啊。<br><br>
<span style="font-weight:bold">&emsp;&emsp;“蠡山沄居”创始人 唐仲磊：</span>每当我劳累下班之后回到家里面吃完晚饭，走出家门，那就是一个很好的风景画面，是一个很和美的一个画卷。<br><br>
<span style="font-weight:bold">&emsp;&emsp;河南游客 魏琰笑：</span>就是能感受到人与人之间那种平等、那种宁静，人与自然也是一样的，因为大自然它本身就有治愈一切的能量。<br><br>
<span style="font-weight:bold">&emsp;&emsp;“稻颂乾元”矿坑营地创始人 谢逊：</span>和美乡村它是一种载体。乡村有乡村的一种乡愁或者情怀在里面。它也很适合现在的年轻人投身到乡村里面，找到他的一些归属感，通过他的一种付出去找到自己的人生的意义跟价值。<br><br>
<span style="font-weight:bold">&emsp;&emsp;上海游客 才华：</span>摘菜、洗菜，然后自己剁自己包。有带小朋友过来让他们来体验一下。<br><br>

<span style="font-weight:bold">&emsp;&emsp;【央视短评】此心安处是吾乡</span><br><br>
&emsp;&emsp;很多中国人的骨子里都有一份乡土情结。75年前新中国刚成立的时候，10个中国人里有9个人是农民。今天的广大农村地区，依然是近5亿农民常住的家园。农村美不美，农民生活好不好，直接影响着中国式现代化的成色和质量。建设宜居宜业的和美乡村，既要有田园风光的外在美，也要有幸福生活的内在美。既要留住乡风乡韵乡愁，也要让广大农民群众就地过上现代文明生活，实现由表及里、内外兼修、形神兼备的乡村大美。这样的乡村，发展机遇天地广阔，自然能吸引人；这样的乡村，田园宜居环境优美，也就能留住人。<br><br>
